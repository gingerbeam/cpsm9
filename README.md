# Quick start

## Install PBC Library

```bash
wget https://crypto.stanford.edu/pbc/files/pbc-0.5.14.tar.gz
tar zxvf pbc-0.5.14.tar.gz
cd pbc
./configure
make
sudo make install
sudo ldconfig
```

## Build

```bash
mkdir build && cd build
cmake ..
make
```

## Run

```bash
cd build
./xxx_test
```
